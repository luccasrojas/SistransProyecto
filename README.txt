----PROCESO DE INSTALACION DE LA APLICACIÓN----
1. Primero descargar la aplicacion del respositorio: https://github.com/DISC-isis2304-ST/C-09.git
2. Abrir el proyecto y modificar los datos de conexión en el archivo persistence.xml para conectar a la BD.
3. Abrir SQL y correr la creación de las tablas, información que puede ser encontrada en la carpeta data del proyecto.
4. En SQL developer poblar las tablas utilizando las sentencias que se enucentran en la carpeta data del proyecto.
5. Finalmente, desde su editor de código correr el proyecto, especificamente en src/main/java/InterfazHotelandesApp.java 
6. Utilizar las funcionalidades que brinda la app desde la barra de navegación.
